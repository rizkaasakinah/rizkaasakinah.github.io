Tentang

Saya tertarik di bidang web hoster. Saya memiliki skill di bidang digital creativity termasuk web hosting. Saya adalah seorang yang dapat belajar otodidak, bekerja tim, motivated. Dan saya menyukai semua bidang yang ditawarkan (Designer, Copy writer, Web hosting) karena saya mempunyai skill di ketiganya.

Selama dua tahun, saya mempunyai pengalaman dalam hal pengelolaan sosial media (website) dan content creator di sebuah organisasi kampus. Saya membuat project website , dengan jobdesk mengelola web, design , dan sebagai copy writer. Saya pernah mendapatkan pelatihan dan sosialisasi di tahun 2019 sebagai admin website oleh fasilitator BSI UMY.
